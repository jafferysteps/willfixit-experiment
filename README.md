# Will Fix It Style Experiment

A test landing page design modeled after **WillFixIt.com** (San Antonio, TX).

## Key Design Patterns Applied:
- **Hero Form Integration**: Places the scheduler form directly on the main banner overlay.
- **Corporate Blue & Bright Orange Brand Theme**: Uses deep corporate blue (`#0f2b5c`) for trust/stability and bright orange (`#f26522`) for callouts, tags, and urgent CTAs.
- **Coupons Grid Section**: High-converting service coupon cards with custom borders and discount tags.
- **Montserrat & Poppins Typography**: Clean professional typography.
